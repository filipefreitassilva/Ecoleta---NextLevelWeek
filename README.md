## Instalação

Configurar Projeto Nodejs
```sh
npm init -y
```

Instalar Express (Servidor)
```sh
npm install express
```

Instalar NodeMon
```sh
npm install nodemon
```

Instalar Nunjucks
```sh
npm install nunjucks
```
